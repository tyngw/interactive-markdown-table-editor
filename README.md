# Markdown Table Editor

A VSCode extension that allows you to edit Markdown tables with a spreadsheet-like interface.

## Features

- Edit Markdown tables directly in preview mode
- Spreadsheet-like UI with grid lines, row numbers, and column headers
- Add/remove rows and columns
- Sort columns by clicking headers
- Drag and drop to reorder rows and columns
- Keyboard navigation (arrow keys, Tab, Enter)
- Right-click context menu integration

## Development

This extension is currently in development.

### Building

```bash
npm install
npm run compile
```

### Testing

```bash
npm test
```

## Requirements

- VSCode 1.74.0 or higher

## Extension Settings

This extension contributes the following settings:

* `markdownTableEditor.enable`: Enable/disable this extension
